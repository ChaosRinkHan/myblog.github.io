文字背景色块

添加

打开themes/next/source/css/_custom 下的 custom.styl 文件,添加自定属性样式

想添加自定义颜色，修改最后的backgroud-color即可。

使用

在md正文中如下格式：

<span id="inline-blue"> Text here </span>

就可使用了。

效果：

Text Text Text Text

使用FA图标

在md正文中使用

<i class="fa fa-subway"></i>

来调用FA图标。第二个fa后单词改为需要的图标名称即可。
